# AI Rules



Last updated: 03.07.2026



Do not rewrite the whole application.

Only change the files that are explicitly requested.

Do not change working code unless necessary.

Work in small steps.

Before writing code, explain which files will be changed.

Implement only one module or one feature at a time.

Keep the existing Java style.

Do not introduce frameworks.

Do not create a GUI.

The application remains a Java CLI application.

All code must stay simple and understandable.

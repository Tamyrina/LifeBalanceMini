\# Future Extensions



Last updated: 04.07.2026



\## Planned Features



\### Repeating Tasks



The Task model already contains a repeat attribute.



At the moment, this attribute is stored but not evaluated.



A future version could automatically create recurring tasks, for example:



\- Daily

\- Weekly

\- Monthly

\- Yearly



\---



\### Overdue Task Notifications



Display overdue tasks on the Dashboard.



\---



\### Project Notes



Allow users to store notes for each project.



\---



\### Project Colors



Assign colors to projects for easier identification.



\---



\### Task Search



Search tasks by title, project or date.



\---



\### Task Filtering



Filter tasks by:



\- project

\- priority

\- completion status



\---



\### Archive Completed Projects



Move completed projects to a separate archive instead of displaying them in the active project list.
